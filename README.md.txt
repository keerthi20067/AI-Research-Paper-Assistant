# AI Research Paper Assistant

## Description

AI Research Paper Assistant is a smart tool that helps users understand research papers quickly by generating summaries and answering questions about the paper using AI.

## Features

* Upload research papers (PDF)
* Generate AI-based summaries
* Ask questions about the research paper
* Extract important insights
* Simple and interactive UI

## Tech Stack

* Python
* Streamlit
* OpenAI API
* LangChain
* FAISS

## Installation

1. Clone the repository
   git clone https://github.com/yourusername/AI-Research-Paper-Assistant.git

2. Navigate to the project directory
   cd AI-Research-Paper-Assistant

3. Install dependencies
   pip install -r requirements.txt

4. Run the app
   streamlit run app.py
